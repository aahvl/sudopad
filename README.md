# SudoPad

## A cyberpunk themed sayopad inspired device.

Made for Geometry Dash, Osu, and a ton of other rhythm inspired games, aswell as shortcuts.

Consists of: A OLED Screen, Rotary Encoder and 3 Cherry MX Switches.
